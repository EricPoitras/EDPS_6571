Predicting Titanic Survival

An example static dashboard on a supervised machine learning problem - classifying survival based on passenger characteristics. The training and test set is available at Titanic: Machine Learning from Disaster from Kaggle. The RapidMiner 9.3 implementation of the decision tree algorithm was fitted to the data.
